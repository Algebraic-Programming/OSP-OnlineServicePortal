    About OneStopParallel
    OneStopParallel is an open-source toolkit developed by the Computing Systems Laboratory at the Zurich Research Center of Huawei Technologies Switzerland AG. It implements a suite of advanced scheduling algorithms for parallel computing systems under the Bulk Synchronous Parallel (BSP) model, intelligently assigning tasks to processors while balancing computational load, respecting memory constraints, and minimizing communication overhead.

    🔗 View the current source code on GitHub

    📄 Related publications:
    Papp et al., Efficient Multi-Processor Scheduling in Increasingly Realistic Models , SPAA 2024
    Böhnlein et al., Efficient Parallel Scheduling for Sparse Triangular Solvers , arXiv 2025
